# Research Posts

This directory will contain research posts generated by the Tactics Journal research pipeline.

## Permalink structure

Research posts should use: `/research/:year/:month/:day/:title/`

Add this to each post's front matter:

```yaml
permalink: "/research/YYYY/MM/DD/slug/"
layout: post
```

## Post format

Posts generated by the pipeline should follow standard Jekyll front matter:

```yaml
---
layout: post
date: YYYY-MM-DD HH:MM UTC
title: "Post title"
categories:
  - "Research"
permalink: "/research/YYYY/MM/DD/slug/"
---
```
